• path: File or directory sub-path to search within. * and ? act as shell wildcards.
  • ext: Filename extension: ext:cpp
  • regexp: Regular expression. Examples: regexp:(?i)\bs?printf regexp:"(three|3) mice"
  • loc: Things declared/used at a position loc:foo.c:42
  • qualname: Fully qualified variable/function name
  • type: Usages of type
  • index: RegExp query against the raw index data
